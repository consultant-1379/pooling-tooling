Overview

This page describes the pools administration features in RPT.

It describes the purpose of these features and how they work.

Purpose

The purpose of having pools administration section in RPT is to provide the ability to:

  • Add Pool
  • Remove Pool

Add Pool

Steps

1 Enter pool name in the search box

  • Please note that this is case sensitive.

2 Click on the 'Add Pool' button

  • If the pool exists dialog box will be populated stating "Pool already exists in RPT".
  • If the pool does not exist, Add Pool form will be populated.

3 Please enter the pool's properties

  • Pool Properties Description
    Creator Area Select an appropriate area to which the pool belongs.
    Creator Name This field will be populated with your username.

4 Click on 'Add Pool'

Remove Pool

Steps

1 Enter pool name in the search box

  • Please note that this is case sensitive.

2 Click on the 'Remove Pool' button

  • If the pool does not exist, a dialog box will pop up stating that pool does not exist in RPT.
  • If the pool exists, the Remove Pool form will be opened.

3 Re-enter the pool's name

  • To confirm, please re-enter the name of the pool to be removed.
  • The submit button will only be enabled if the name of the pool has been re-entered.

4 Click on the 'Submit' button to remove the pool
